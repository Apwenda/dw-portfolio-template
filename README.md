# DW Portfolio Template

Modern personal portfolio website template built with HTML, Tailwind CSS, and Alpine.js.

## Features

- Responsive Design
- Dark Mode
- SEO Friendly
- Portfolio Showcase
- Testimonials Carousel
- WhatsApp CTA
- Fast Performance
- Open Source (MIT License)

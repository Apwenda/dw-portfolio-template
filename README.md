# DW Portfolio Template

Modern personal portfolio website template built with HTML, Tailwind CSS, and Alpine.js. Designed for developers, freelancers, creatives, and professionals who want a clean, responsive, and modern online presence.

## ✨ Features

* Fully Responsive Design
* Dark Mode Support
* SEO Friendly Structure
* Portfolio Showcase Section
* Testimonials Carousel
* WhatsApp Call-to-Action
* Modern UI/UX Design
* Fast Loading Performance
* Open Source (MIT License)

## 🚀 Live Demo

https://apwenda.github.io/dw-portfolio-template/

## 🛠️ Built With

* HTML5
* Tailwind CSS
* Alpine.js
* Font Awesome

## 📦 Installation

1. Clone this repository:

```bash
git clone https://github.com/apwenda/dw-portfolio-template.git
```

2. Open the project folder:

```bash
cd dw-portfolio-template
```

3. Open `index.html` in your browser or deploy it to your preferred hosting platform.

# DW Portfolio Template

Template website portofolio modern yang dibangun menggunakan HTML, Tailwind CSS, dan Alpine.js. Dirancang untuk developer, freelancer, kreator, dan profesional yang ingin memiliki website pribadi yang bersih, responsif, dan modern.

## ✨ Fitur

* Desain Responsif di Semua Perangkat
* Dukungan Dark Mode
* Struktur SEO Friendly
* Showcase Portofolio
* Carousel Testimoni
* Tombol WhatsApp Call-to-Action
* UI/UX Modern
* Performa Cepat
* Open Source (MIT License)

## 🚀 Demo

https://apwenda.github.io/dw-portfolio-template/

## 🛠️ Dibangun Dengan

* HTML5
* Tailwind CSS
* Alpine.js
* Font Awesome

## 📦 Instalasi

1. Clone repository:

```bash
git clone https://github.com/apwenda/dw-portfolio-template.git
```

2. Masuk ke folder project:

```bash
cd dw-portfolio-template
```

3. Buka file `index.html` menggunakan browser atau deploy ke layanan hosting pilihan Anda.
